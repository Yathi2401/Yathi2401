#### 🔩 [Smart Pedestrian Crossing Controller](https://github.com/Yathi2401/Smart-pedestrian-crossing-controller-using-Allen-Bradley-PLC-and-RSLogix-500.)

A state-based ladder logic controller developed using **RSLogix Micro Starter Lite** and simulated with **RSLogix Emulate 500**.

The controller manages vehicle and pedestrian traffic signals using minimum-green, yellow, red-clearance and walk timers. It also includes pedestrian request storage, night-mode flashing, emergency override and fault detection.
